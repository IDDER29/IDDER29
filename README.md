## ❯ whoami

I'm **Idder** — a full-stack developer from **Agadir, Morocco**. I ship production web apps and
**Shopify** storefronts for real businesses, and I'm deep in the **42 Network** curriculum at
**1337**, building low-level **C** from scratch: a custom `printf`, a sorting engine, my own
