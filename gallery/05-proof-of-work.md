## Built from first principles

**1337 Systems Suite** · [repos (C, sorted by stars)](https://github.com/IDDER29?tab=repositories&language=c)

Every program below was written without external libraries. Each one required understanding what the standard library actually does before being allowed to use it:

| Project | What it is |
|---|---|
| `ft_printf` | `printf` from scratch — variadic args, format parsing, edge cases |
| `push_swap` | Sorting algorithm selection + implementation, minimum operations |
| `libft` | A standard library re-implementation: memory, strings, lists |
| `get_next_line` | File descriptor reading with static state across calls |
| `minishell` | A POSIX-compatible shell: pipes, redirections, signals, builtins |
